帮助

  • man 命令用于查看命令的手册页，提供了关于命令的详细信息，包括命令的选项、用法和示例。手册页通常按照章节组织，可以使用 man 命令查看不同章节的手册页。

  • help 命令是 Bash shell\text {Bash shell}Bash shell 的内置命令，用于获取 shell\text {shell}shell 内置命令的简要帮助信息。它通常不提供外部命令（例如系统命令）的帮助信息。

  • info 命令也用于查看命令的帮助文档，但与 man 不同，info 提供了更为交互式和结构化的帮助信息。info 帮助信息通常以节点（node\text {node}node）的形式组织，用户可以导航到不同的节点查看相关信息。

”一切皆文件“

pwd

pwd：用于显示当前工作目录的路径。

ls

ls ：用于列出目录内容，即显示当前工作目录中的文件和子目录。

常见选项：

  • -a： 列出所有文件和文件夹，包括以.开头的隐藏文件。

  • -l： 以长格式显示文件列表，包括文件权限、所有者、大小等详细信息。

  • -r：反转排序顺序，以逆序列出文件和文件夹。

  • -t：按修改时间排序，最新修改的文件在前。

  • -R： 递归列出子目录的内容。
